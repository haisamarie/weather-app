# 天気予報アプリ

city をフォームで入力すると天気と気温を OpenWeatherMap API から取得して表示する

## 🏗 技術スタック

| 分類           | 技術        |
| -------------- | ----------- |
| フレームワーク | React       |
| 言語           | TypeScript  |
| スタイル       | CSS modules |
| デプロイ       | Vercel      |

## 📦 セットアップ

```
npm install
npm run dev
```
